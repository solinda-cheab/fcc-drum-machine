# Drum Machine

A modern Drum Machine built with React for the FreeCodeCamp Front End Development Libraries Certification. Play drum sounds using your keyboard or mouse, with real-time audio feedback and an interactive interface.

## Features

-  9 interactive drum pads
-  Keyboard support (Q, W, E, A, S, D, Z, X, C)
-  Mouse click support
-  High-quality drum sounds
-  Display shows the currently playing sound
-  Instant audio playback
-  Responsive design
-  Passes all FreeCodeCamp user stories

##  Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/fcc-drum-machine.git
```

Navigate to the project folder:

```bash
cd fcc-drum-machine
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

---

##  Controls

| Key | Sound |
|------|-------|
| Q | Heater 1 |
| W | Heater 2 |
| E | Heater 3 |
| A | Heater 4 |
| S | Clap |
| D | Open HH |
| Z | Kick n' Hat |
| X | Kick |
| C | Closed HH |

You can also click the on-screen drum pads to trigger the sounds.
